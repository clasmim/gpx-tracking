# Running LaSafor
A GPX Tracking application with the name "Running LaSafor". Ability to create an account, upload GPX files, view history, displaying paths and adding annotations.

## Objective:
Upload GPX files from a tracking device and observe the path and statistical values at a simple glance. Also add annotations and record the user's history. 

## Usage and execution:
In order to compile/run the application the JavaFX library is necessary, it can be added to the `\lib\` folder.

Simply create an account (they are saved locally) and upload a GPX file to start using the app.

Most of the information about usage is within the `About` tab in the top right of the Dashboard once you are logged in.
